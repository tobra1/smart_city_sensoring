Smart City Sensoring

gprs_sensor_box contains source code.
Libraries contains the libraries used by the code. These needs to be added to the 
"C:\Program Files (x86)\Arduino\libraries" folder.

Port   Sensor
A0     Grove - Sound Sensor
D8     Grove - Dust Sensor
I2C    Grove - Barometer
I2C    Grove - Sunlight Sensor
I2C    Grove - Multichannel Gas
I2C    Grove - Temperature&Humidity (HDC1000)
UART   Grove - CO2 Sensor
